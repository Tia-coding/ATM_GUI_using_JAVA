# ATM Simulation System (Java Swing)

The ATM Simulation System is a desktop-based banking application that replicates the functionality of a real-world ATM.
It is developed using Java Swing and AWT, focusing on Object-Oriented Programming (OOP), GUI design, and data handling using collections.
The system allows users to authenticate securely and perform multiple banking operations such as balance inquiry, withdrawal, and deposit, while maintaining transaction records dynamically.

Features:

 Secure PIN-based authentication
 Balance inquiry
 Cash withdrawal & deposit
 Transaction receipt generation
 Multi-screen navigation using CardLayout
 Input validation and error handling
 
Tech Involved:

 Java (Core + OOP)
 Java Swing & AWT
 CardLayout (UI navigation)
 HashMap (user data)
 ArrayList (transaction history)

Architecture:

GUI Layer: Swing components for user interaction
Logic Layer: Handles authentication and transactions
Data Layer: In-memory storage using HashMap and ArrayList


Workflow:

WELCOME SCREEN → PIN ENTRY → MAIN MENU → TRANSACTION → RECEIPT → EXIT SCREEN

How to Run:

1. Clone the repository
2. Open in Eclipse
3. Run "SimpleATM_final.java"
